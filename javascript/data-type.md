# Data Type

## \# Data Type

All of data in JavaScript has a type. The data type is also stored in memory in binary. Data is largely divided into primitive types and object types.

* Primitive data type

  * Boolean
  * Null
  * Undefined
  * Number
  * String
  * Symbol \(ES6\)

* Object/reference type
  * Object
    * Function
    * Arrays
    * Object
